Welcome to Bib Detector

Bib Detector is an application designed to streamline the process of analyzing race finishes. Upload a video of race finishes, and Bib Detector will generate a CSV file sent to your email, containing finishers listed by their bib number and gender.

Bib Detector was created as a project for Informatics 4000. It combines YOLOv8n, YOLOS, ResNet50, and OCR for object detection, image classification, and number recognition. The GitHub for this project can be found here.
